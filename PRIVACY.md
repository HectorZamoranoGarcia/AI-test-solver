# Privacy Policy — AI Test Solver

**Last updated:** May 4, 2025

## Data Collection

AI Test Solver does **not** collect, store, or transmit any personally identifiable information.

## Data Used

The extension reads the **visible text content** of the active webpage solely to send it to the Google Gemini API for the purpose of answering multiple-choice questions. This text is not stored, logged, or shared with any party other than Google's Generative Language API.

## API Key Storage

The Google Gemini API key entered by the user is stored exclusively in `chrome.storage.sync` within the user's own browser. It is never transmitted to any server other than `generativelanguage.googleapis.com`.

## Third Parties

The only third-party service used is the **Google Gemini API** (`generativelanguage.googleapis.com`). Google's own privacy policy applies to data processed through their API.

## No Tracking

This extension contains no analytics, telemetry, advertising, or user tracking of any kind.
